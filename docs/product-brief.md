# Product Brief

## Формат
- Single Page Application (одна сторінка).
- Основний сценарій: мобільний портретний режим.
- Підтримка адаптації для планшета/десктопа.
- Вертикальне вікно (екран-кадр), прокрутка сторінки вимкнена.

## Структура екрана
1. `Header` (панель керування):
   - перемикач режимів: `Classic` / `Beads`;
   - відображення раунду;
   - кнопка скидання (опціонально з підтвердженням);
   - службові кнопки (пізніше: налаштування, звук/вібрація тощо).
2. `Body` (робоча зона):
   - режим `Classic`: кнопки `+` і `-`, дисплей числа;
   - режим `Beads`: інтерактивна вертикальна стрічка бусин.

## Правила рахунку
- Один раунд = `108` кроків.
- Досягнення 108 автоматично переводить у наступний раунд.
- Логіка декременту підтримує повернення в попередній раунд.

## UX-пріоритети
- Великі touch-зони для мобільного.
- Мінімум зайвих дій: один жест/тап = одна дія.
- Плавна та передбачувана анімація без лагів.
- Відсутність випадкового системного скролу під час жестів.

## Майбутні розширення
- Інтеграція кастомного зображення клікера з інтерактивними зонами `+`, `-`, дисплей.
- Збереження стану в `localStorage`.
- Лічильник сесій/історія.
