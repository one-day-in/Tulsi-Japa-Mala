# Open Questions

## Зафіксовані рішення
1. Дисплей показує кроки `1..108`.
2. `Reset` скидає весь прогрес: `round=1`, `countInRound=0`.
3. Режим `Beads`: один завершений жест = один крок.
4. Мультикрок за довжиною свайпу вимкнений.
5. Стан обов'язково зберігається у `localStorage`.

## Додаткові уточнення на майбутнє
1. Візуальний стиль бусин: реалістичні чи мінімалістичні.
2. Чи додаємо вібрацію/звук на крок.
3. Чи потрібна мультимовність інтерфейсу.

## Технічні примітки
- Для iOS Safari варто тестувати `100dvh` + safe-area (`env(safe-area-inset-*)`).
- Для жестів важливо уникнути конфлікту з pull-to-refresh браузера.
